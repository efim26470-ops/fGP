# Retro Turbo GP

Псевдо-3D ретро-гонка в стиле аркадных 8/16-bit racing games. Проект сделан без сторонних баз данных, серверов и внешних ассетов: достаточно открыть `index.html` или выложить папку на GitHub Pages.

## Что уже есть

- Выбор авто, трассы и сложности.
- Управление стрелками и WASD.
- Мобильные экранные кнопки газ/тормоз/поворот.
- Магазин авто за монеты, которые собираются на дороге.
- Разные характеристики машин: максимальная скорость, разгон, торможение, управляемость, сцепление, стабильность.
- Разные трассы: город, классический зелёный трек, серпантин, пустынная скоростная трасса и снежное кольцо.
- HUD: скорость, тахометр, передачи, одометр, дистанция, таймер, урон, поворотники, монеты, мини-карта.
- Локальное сохранение через `localStorage`: монеты, купленные машины, выбранные настройки, рекорды и общий одометр.
- PWA-манифест и service worker для простого запуска с GitHub Pages и добавления на экран iPhone.

## Как выложить на GitHub Pages

1. Создай репозиторий, например `retro-turbo-gp`.
2. Загрузи в корень репозитория файлы из этой папки:
   - `index.html`
   - `styles.css`
   - `game.js`
   - `manifest.webmanifest`
   - `sw.js`
   - `icon.svg`
3. В GitHub открой **Settings → Pages**.
4. В разделе **Build and deployment** выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохрани и подожди, пока GitHub выдаст ссылку.

## Настройка игры

Все основные данные лежат в начале `game.js`:

- `CARS` — машины, цены и характеристики.
- `TRACKS` — трассы, визуальный стиль, количество кругов, сцепление, трафик, монеты и набор поворотов.
- `DIFFICULTIES` — сложность, награды, плотность трафика и урон.

Чтобы добавить новую машину, скопируй один объект из `CARS`, измени `id`, `name`, `price`, `color`, `maxSpeed`, `accel`, `handling`, `grip` и т.д.

Чтобы добавить новую трассу, скопируй объект из `TRACKS`. Поле `pattern` задаёт дорожную геометрию: `[количество_сегментов, сила_поворота]`. Например `[40, 0]` — длинная прямая, `[16, .7]` — правый поворот, `[16, -.7]` — левый поворот.

## Важно

Прогресс хранится только в браузере игрока. Если очистить данные сайта или открыть игру в другом браузере, прогресс начнётся заново.
