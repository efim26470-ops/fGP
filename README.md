# Q3 Retro Rally HTML v9

Версия для GitHub Pages без сервера и базы данных. Прогресс хранится локально в `localStorage`.

## Что исправлено в v9

- трасса заново отрисовывается стабильной псевдо-3D проекцией и больше не исчезает под машиной;
- отключена сильная тряска экрана;
- машина привязана к полотну и не «левитирует»;
- управление сглажено: есть руль, боковая инерция, центробежный вынос на поворотах и штраф за обочину;
- скорость ощущается через движение разметки, монет, трафика и масштаб объектов;
- превью машин в гараже берутся из папки `assets/cars`;
- HUD, мини-карта, монеты, покупка авто и выбор трасс сохранены.

## Управление

- `W` / `↑` — газ
- `S` / `↓` — тормоз
- `A` / `←` — влево
- `D` / `→` — вправо
- `Space` — ручник
- `P` — пауза

Русская раскладка поддерживается.

## Публикация на GitHub Pages

1. Распакуй архив.
2. Загрузи всё содержимое папки в корень репозитория.
3. Включи GitHub Pages: Settings → Pages → Deploy from branch → main / root.
4. После обновления открой сайт с `?v=9` на конце ссылки или очисти кэш, если браузер держит старую сборку.
