# Q3 Retro Rally HTML v8

Статическая HTML/Canvas-гонка для GitHub Pages. Работает без базы данных и сервера, прогресс хранится в `localStorage`.

## Что исправлено в v8

- Пересобрана проекция трассы: дорога теперь стабильно рисуется как псевдо-3D полотно, без ощущения, что машина летит отдельно от трассы.
- Машина визуально привязана к дорожному полотну и не левитирует над сценой.
- Управление сглажено: руль не резкий, но авто заметно смещается по дороге.
- Скорость откалибрована: HUD показывает адекватные км/ч, а разметка, объекты и фон дают более понятное ощущение движения.
- Уменьшены визуальные разрывы/полосы между сегментами дороги.
- Превью машин в гараже оставлены через `assets/cars`, меню использует локальные изображения.
- Оставлены выбор авто, трассы, сложности, покупка машин за монеты, локальный прогресс, HUD, мини-карта, трафик и монеты.

## Управление

- `W` / `↑` — газ
- `S` / `↓` — тормоз
- `A` / `←` — влево
- `D` / `→` — вправо
- `Space` — ручник / дрифт
- `P` — пауза

## GitHub Pages

1. Распакуй архив.
2. Загрузи все файлы в корень репозитория.
3. Включи GitHub Pages: `Settings` → `Pages` → `Deploy from branch` → `main` → `/root`.
4. Открывай игру по ссылке Pages.

Нативные файлы Q3Rally (`.dylib`, macOS-бинарники, SDL/OpenGL) нельзя напрямую запустить на GitHub Pages. Для браузера нужен исходный код и сборка в WebAssembly. Поэтому v8 — это самостоятельный Canvas-движок, использующий локальные ассеты и идею аркадной псевдо-3D гонки.
