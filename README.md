# Q3 Retro Rally HTML

HTML-версия псевдо-3D гонки для GitHub Pages. Работает без сервера, базы данных и сторонних библиотек. Прогресс сохраняется локально в `localStorage`.

## Что взято из загруженного пакета

Из присланных файлов Q3Rally использованы детали, которые реально можно запустить в браузере:

- иконки машин из `q3rpak0.pk3`;
- HUD-графика спидометра/стрелки из `q3rpak0.pk3`;
- звуки старта, чекпоинта, скольжения и удара из `q3rpak0.pk3`;
- превью трасс из `levelshots`.

Нативные файлы `q3rally`, `*.dylib`, `libSDL2-2.0.0.dylib` являются macOS Mach-O бинарниками. GitHub Pages и браузер не могут напрямую запускать такие файлы. Для настоящего запуска оригинального движка в HTML нужен исходный код и сборка через Emscripten/WebAssembly. Поэтому здесь сделана отдельная HTML/Canvas-реализация, вдохновлённая этим пакетом и NES-псевдо-3D гонками.

## Запуск локально

Открой `index.html` в браузере. Для корректной работы Service Worker/PWA лучше запускать через локальный сервер:

```bash
python3 -m http.server 8080
```

Затем открыть `http://localhost:8080`.

## Публикация на GitHub Pages

1. Загрузи все файлы из папки в репозиторий.
2. В настройках репозитория включи **Pages**.
3. Source: `Deploy from branch`, branch `main`, folder `/root`.
4. Открой ссылку Pages.

## Управление

- `W` / `↑` — газ;
- `S` / `↓` — тормоз;
- `A` / `←` — влево;
- `D` / `→` — вправо;
- `Space` — ручник/дрифт;
- `P` — пауза.

На телефоне есть экранные кнопки.

## Игровые возможности

- выбор авто, трассы и сложности;
- покупка машин за монеты;
- разные характеристики машин: скорость, разгон, сцепление, масса, управляемость;
- разные трассы: прямые/повороты/сцепление/визуальный стиль;
- монеты на дороге;
- трафик и столкновения;
- HUD: скорость, передачи, тахометр, урон, круг, время, дистанция, одометр, поворотники, мини-карта;
- PWA-манифест и офлайн-кэш.
