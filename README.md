# Retro Grand Prix 8-bit

Статичная псевдо-3D гонка для GitHub Pages. Работает без сервера, без базы данных и без внешних библиотек. Прогресс сохраняется в `localStorage` браузера.

## Что внутри

- `index.html` — страница игры.
- `styles.css` — интерфейс, адаптация под ПК и телефон.
- `game.js` — движок, физика, трассы, гараж, HUD, сохранение.
- `manifest.webmanifest`, `sw.js`, `icon.svg` — PWA-режим и иконка для добавления на главный экран.

## Возможности

- Выбор авто, трассы и сложности.
- Покупка авто за монеты, собранные на дороге.
- Разные характеристики машин: максимальная скорость, разгон, тормоза, руль, сцепление и масса.
- Разные трассы: прямые, быстрые дуги, городские связки, горные S-повороты.
- Управление: WASD и стрелки.
- Мобильные кнопки управления.
- HUD: скорость, передачи, тахометр, круг, время, дистанция, одометр, урон, поворотники, мини-карта.
- Локальное сохранение прогресса.

## Публикация на GitHub Pages

1. Создай новый репозиторий или открой существующий.
2. Загрузи все файлы из архива в корень репозитория.
3. Открой `Settings` → `Pages`.
4. В `Build and deployment` выбери `Deploy from a branch`.
5. Branch: `main`, folder: `/root`.
6. Нажми `Save`.
7. Через 1–2 минуты игра будет доступна по ссылке GitHub Pages.

## Важно

Это оригинальная игра, визуально и механически вдохновленная 8/16-bit аркадными гонками. В ней не используются ассеты, названия и код Ferrari Grand Prix Challenge.
