# Q3 Retro Rally HTML v7

Статическая HTML/Canvas-гонка для GitHub Pages. Работает без базы данных и сервера, прогресс хранится в `localStorage`.

## Что исправлено в v7

- Управление переписано: клавиши теперь считываются через физические `e.code` и через символы `e.key`, поэтому WASD работает и на английской, и на русской раскладке.
- Машина теперь реально смещается по полосе, а не остаётся приклеенной к центру экрана.
- Камера только частично следует за машиной: видно, куда едет авто, и пропадает ощущение, что руль не работает.
- Боковая физика сделана более аркадной и предсказуемой: плавный руль, понятный возврат, сцепление, вынос наружу поворота, обочина и ручник.
- Добавлен сброс зажатых клавиш при потере фокуса окна, чтобы управление не залипало.
- Оставлены выбор авто, трасс, сложности, покупка машин за монеты, локальный прогресс, мини-карта, HUD, трафик и монеты.

## Управление

- `W` / `↑` — газ
- `S` / `↓` — тормоз
- `A` / `←` — влево
- `D` / `→` — вправо
- `Space` — ручник / дрифт
- `P` — пауза

## GitHub Pages

1. Распакуй архив.
2. Загрузи все файлы в корень репозитория.
3. Включи GitHub Pages: `Settings` → `Pages` → `Deploy from branch` → `main` → `/root`.
4. Открывай игру по ссылке Pages.

Нативные файлы Q3Rally (`.dylib`, macOS-бинарники, SDL/OpenGL) нельзя напрямую запустить на GitHub Pages. Для браузера нужен исходный код и сборка в WebAssembly. Поэтому v7 — это самостоятельный Canvas-движок, использующий структуру, ассеты и идею из пакета, но не пытающийся загрузить нативные `.dylib` в браузер.
