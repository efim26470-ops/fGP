# Retro Turbo GP

Псевдо‑3D ретро‑гонка для GitHub Pages без серверов и баз данных. Весь прогресс сохраняется локально в браузере через `localStorage`.

## Что есть

- Выбор автомобиля, трассы и сложности.
- Покупка машин за монеты, которые собираются прямо на дороге.
- Разные характеристики машин: максимальная скорость, разгон, тормоза, сцепление, масса, стабильность.
- Разные трассы: город, классический зелёный трек, серпантин, пустынная скоростная трасса, ледяное кольцо.
- Улучшенная аркадная физика: инерция, снос наружу поворота, занос, сцепление покрытия, урон за вылеты и столкновения.
- HUD в стиле старых гоночных автоматов: скорость, передача, тахометр, спидометр, поворотники, урон, дистанция, одометр, таймер, монеты, мини-карта.
- Управление: стрелки и WASD.
- Сенсорные кнопки для телефона.
- PWA-файлы: `manifest.webmanifest`, `sw.js`, `icon.svg`.

## Управление

- `W` или `↑` — газ.
- `S` или `↓` — тормоз.
- `A/D` или `←/→` — руль.
- `P` — пауза.
- `R` — рестарт гонки.

## Установка на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи в корень репозитория все файлы из этой папки:
   - `index.html`
   - `styles.css`
   - `game.js`
   - `manifest.webmanifest`
   - `sw.js`
   - `icon.svg`
3. Открой Settings → Pages.
4. В Source выбери ветку `main` и папку `/root`.
5. Открой ссылку GitHub Pages.

## Важно

Игра не использует внешние API, базы данных, CDN и сторонние изображения. Она запускается просто по ссылке, а сохранение работает только на устройстве пользователя.
