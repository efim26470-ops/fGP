# Q3 Retro Rally HTML v6

Статическая HTML/Canvas-гонка для GitHub Pages. Работает без базы данных и сервера, прогресс хранится в `localStorage`.

## Что исправлено в v6

- Полностью переписан псевдо-3D рендер дороги: машина теперь визуально закреплена на полотне, а не летает отдельно от трассы.
- Переделана физика: плавный руль, боковая инерция, сцепление, вынос наружу поворота, штраф за обочину, ручник/дрифт.
- Скорость стала адекватнее: цифры на HUD показывают нормальные км/ч, а ощущение скорости создают быстрые сегменты дороги, разметка, объекты и speed-lines.
- Убрано лишнее полосение: меньше мелких горизонтальных полос и меньше субпиксельных щелей между сегментами.
- Превью машин в меню берутся из `assets/cars`.
- Оставлены выбор авто, трасс, сложности, покупка машин за монеты, локальный прогресс, мини-карта, HUD, трафик и монеты.

## Управление

- `W` / `↑` — газ
- `S` / `↓` — тормоз
- `A` / `←` — влево
- `D` / `→` — вправо
- `Space` — ручник / дрифт
- `P` — пауза

## GitHub Pages

1. Распакуй архив.
2. Загрузи все файлы в корень репозитория.
3. Включи GitHub Pages: `Settings` → `Pages` → `Deploy from branch` → `main` → `/root`.
4. Открывай игру по ссылке Pages.

Нативные файлы Q3Rally (`.dylib`, macOS-бинарники, SDL/OpenGL) нельзя напрямую запустить на GitHub Pages. Для браузера нужен исходный код и сборка в WebAssembly. Поэтому v6 — это самостоятельный Canvas-движок, использующий структуру, ассеты и идею из пакета, но не пытающийся загрузить нативные `.dylib` в браузер.
